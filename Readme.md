# AI Resume Analyzer

An AI-powered resume analyzer built using Streamlit and Python.

## Project Overview

AI-powered Resume Analyzer using Gemini AI.

## Features

- Upload Resume PDF
- Extract Resume Text
- Display Resume Content
- Tech Stack
- AI Analysis
- Skill Detection
- ATS Score
- Career Recommendations

## Tech Stack

- Python
- Streamlit
- Gemini API
- pdfplumber

## Future Improvements

- Job Role Matching
- Resume Comparison
- Export Analysis to PDF
